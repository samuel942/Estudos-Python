## 01 - Simulador de Controle Remoto

Sistema que emula o funcionamento de uma televisão no terminal, focado em gerenciamento de estado e interface CLI (Command Line Interface).

### Conceitos aplicados
* Programação Orientada a Objetos (Classes e métodos).
* Gerenciamento de estado (Ligado/Desligado).
* Lógica de navegação circular para canais.
* Interface visual com a biblioteca Rich.

### Execução
1. Instalação de dependência:
   pip install rich

2. Comando para rodar:
   python 01-Controle-Remoto/ControleRemoto.py

### Comandos do simulador
* @ : Liga
* 0 : Desliga
* `+` / - : Volume
* < / > : Navegação de canais
